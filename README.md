## <div align="center">🛠️ API REFERENCE</div>

### DotNetLoader

```python
from py_dotnet_importer import DotNetLoader

# ╔══════════════════════════════════════════════════════════════╗
# ║  BASIC USAGE                                                 ║
# ╚══════════════════════════════════════════════════════════════╝

loader = DotNetLoader(
    auto_resolve_deps=True,      # Auto-resolve dependencies
    generate_wrappers=False       # Generate .py wrappers
)

# Load DLL
result = loader.load("path/to/library.dll")

# ╔══════════════════════════════════════════════════════════════╗
# ║  ADVANCED: WITH NAMESPACE FILTER                             ║
# ╚══════════════════════════════════════════════════════════════╝

result = loader.load(
    "path/to/library.dll",
    namespace="MyCompany.MyProduct"
)

# ╔══════════════════════════════════════════════════════════════╗
# ║  GET CLASS INSTANCE                                          ║
# ╚══════════════════════════════════════════════════════════════╝

instance = loader.get_class_instance(
    assembly_name="MyLibrary",
    class_name="MyNamespace.MyClass",
    *constructor_args
)
```

### DLLAnalyzer

```python
from py_dotnet_importer import DLLAnalyzer

analyzer = DLLAnalyzer("library.dll")

# Get comprehensive metadata
meta = analyzer.get_metadata()
# {
#     "type": "dotnet",
#     "architecture": "x64",
#     "is_dotnet": True,
#     "dotnet_version": "v4.0.30319",
#     "exports": [...],
#     ...
# }

# Check compatibility
analyzer.check_compatibility()  # Raises ArchitectureMismatchError if incompatible
```

### DotNetInspector

```python
from py_dotnet_importer import DotNetInspector

inspector = DotNetInspector("library.dll")

# List all classes
classes = inspector.get_classes()

# Get full API map
api_map = inspector.generate_api_map()

# Get method signature
sig = inspector.get_method_signature("Namespace.Class", "MethodName")
```
